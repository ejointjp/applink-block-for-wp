iPhone / iPad / Macアプリや音楽トラック、Apple Booksなどの紹介リンクを簡単に作成できます。

== Description ==

iPhoneアプリやiPadアプリ、Macアプリ、Apple BooksやiTunes Musicの音楽トラックを紹介するリッチなウィジェットを提供するカスタムブロックです。

もちろんAppleのアフィリエイトリンク対応です。（と言っても今はアプリなどの購入では報酬もらえないかもですが）

== Screenshots ==

1. アイテムのカテゴリーを選択し、検索ワードを入力すると表示される候補からアイテムを選択すると、アイテムのリンクウィジェットを埋め込むことができます。

2. フロント側の表示はこんな感じ。

3. 管理画面から設定ができます。
